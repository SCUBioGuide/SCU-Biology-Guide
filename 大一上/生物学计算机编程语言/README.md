# 学习方法

杨旭家：Python课，可以网课加上课一起学习，比较基础的编程，建议电子笔记。期末作业是一个编程展示

wjh：这门课对于高中未接触过编程的同学或是电脑只会开关机的同学来说，也是相当有挑战性的。你要是真的希望以后可以以此作为科学数据研究的工具，那么你需要多加练习，不要光说不练假把式。课堂上跟着老师学习各个部分的基本操作，理解每条指令的含义，尝试不同的表达。最后的目标就是融会贯通这些部分去解决一个你自己的问题。这门课在以后的数据处理中大有可为，可以作为你参加数学建模比赛、大学生创新创业大赛的有力工具。
